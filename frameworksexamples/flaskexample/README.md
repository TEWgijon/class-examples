# Deploying the Flask example

1. Create the venv:
    ```cmd 
    python3 -m venv .env
    ```
2. Activate it
    ```cmd 
    .env\Scripts\activate
    ```
3. Install django with pip
    ```cmd
    pip3 install flask
    ```
4. Deploy the server
    ```cmd
    flask run
    ```
